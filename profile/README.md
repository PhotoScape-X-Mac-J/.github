## 📋 Overview

<a href="#photoScapeX">PhotoScape X</a> is a comprehensive all-in-one photo editing application for Mac — combining viewer, editor, batch processor, collage maker, GIF creator, and RAW converter in a single application available free from the Mac App Store, providing a broad photo toolkit for users who want more than basic editing without the complexity or cost of professional tools.

The <a href="#photoScapeX">photo editor module</a> provides color correction, brightness, contrast, saturation, highlight and shadow adjustment, white balance, sharpening, and noise reduction — the core adjustment tools that cover most photo enhancement needs. The straightforward interface presents these as adjustment sliders alongside the image with before-and-after comparison, making results immediately visible and the relationship between adjustments and outcomes clear without requiring technical knowledge of what each control does. The <a href="#photoScapeX">filter and effects library</a> applies creative looks through a browsable collection — vintage film simulations, black and white conversions, portrait enhancing, landscape enhancement, and many other creative filters that apply in a single click and are adjustable in strength. The filter library provides creative starting points without requiring manual adjustment of multiple parameters to achieve a particular look.

The <a href="#photoScapeX">batch editing capability</a> applies the same adjustments and effects to multiple photos simultaneously — processing an entire folder with consistent exposure correction, a consistent filter, a watermark, resizing to a specific output resolution, or any combination of adjustments that need to apply uniformly across a photo set. Batch processing eliminates the per-image repetition of applying identical corrections individually. The <a href="#photoScapeX">collage maker</a> arranges multiple photos into grid layouts, free-form arrangements, and template-based compositions with configurable spacing, borders, and background — creating social media posts, photo book pages, and presentation layouts from multiple images without requiring a separate design application.

<a href="#photoScapeX">GIF animation creator</a> builds animated GIFs from sequences of images — creating animated photo series, stop-motion animations, and looping visual content for social media and web use. <a href="#photoScapeX">Cut-out tool</a> removes backgrounds from photos with selection tools for isolating subjects — enabling photo compositing, sticker creation, and any use case where the subject needs to be separated from the original background.

<a href="#photoScapeX">Text and sticker overlays</a> add typography, graphic elements, frames, and decorative elements to photos — creating finished social media images, greeting cards, and decorated photo presentations within PhotoScape X without additional applications. <a href="#photoScapeX">RAW file support</a> opens and processes RAW files from major camera manufacturers for basic RAW conversion and adjustment before export to JPEG or other output formats.
